Expert Answer

PYTHON PROGRAM

#Dictionary tamil contains English words as key and equivalent Tamil words as values
tamil={"I":"naan","We":"Naam","won":"vendrom","Prayer":"Jebam","victory":"Vetri","means":"endral"}
#Read the phrase from user
english_phrase=input("Enter the English Phrase:")
#Split the phrase as words using split function at places where it sees a space
words=english_phrase.split()
print("Word by Word Translation in Tamil:")
#For every word in the phrase, find the value in dictionary and print on single line
for word in words:
    print(tamil[word],end=" ")

EXPLANATION

Dictionary is used to store the translations. English words being key and foreign language equivalents as values. That would assist in easier translations by looking up for the English word and returning the foreign language equivalents.

LIMITATION

The sentence structure obviously varies from language to language and hence the translation will not be exactly how the sentence is spoken. To think of a better approach, we can find the dictionary equivalent and then work on the grammar and sentence formation part using conditionals.
